What is Clirooo?

- Clirooo = CLI + Metro + oo (?)

Clirooo is an application for Checking Tokyo Metro information by using CLI.

- Fater and Cooler

For experienced CLI users, using CLI is often faster and cooler than using GUI.

- Not only Tokyo Metro

We support not only Tokyo Metro but also Tokyo based railway lines.

- Our Supported Lines

  • Tokyo Metro - All 9 Lines (All commands suppported)

  • Toei Subway - All 4 Lines (scheduled to support)

  • JR EAST - Yamanote Line, Chuo Line, Shonan-Shinjuku Line, Keihin-Tohoku Line (scheduled to support)

  • Keio - Keio Line, Inogashira Line (scheduled to support)
